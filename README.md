# 🩺 Blood Bank – Medical Emergency Management SaaS

Une application web moderne permettant de gérer les urgences sanguines, les stocks et les donneurs.  
Interface premium, design hospitalier, développée avec **Laravel 11**, **TailwindCSS 4**, et **Vite**.

---

## 🚀 Fonctionnalités

### 🩸 Urgences
- Création de demandes urgentes
- Suivi des niveaux de criticité (High / Medium / Low)
- Détails complets d’une urgence
- Interface hospitalière moderne

### 🧊 Stock
- Gestion des poches de sang
- Suivi des quantités disponibles

### 👥 Donneurs
- Liste des donneurs
- Informations médicales essentielles

---
